CSS Project

This project was developed as part of Introduction to HTML CSS and Javascript Course Module 2 project. This includes HTML file to define the basic structure and .css file to set the styles and design a responsive page. Features implemented:

  1. Three breakpoints for desktop, tablet and mobile devices
    • Desktop view for 992px and greater
    • Tablet view between 768px and 991px
    • Mobile View for browser width equal to or less than 767px
  2. It consists of a page heading and 3 sections
  3. Three Layouts:
    • Desktop Layout: all in one row
    • Tablet Layout: the first 2 sections should be in the first row and be of equal size. The 3rd section should be in the second row and take up the entire row by itself.
    • Mobile Layout: each section should take up the entire row
  4. Section title region that is always positioned at the top right corner of the section no matter the view (desktop, tablet or mobile)
